# 一般メンバー向けかんたん追加画面

## URL

`/public`

Discordログイン後、参加しているサーバーのうちBotが管理対象にしているサーバーだけが表示されます。

## 追加できるもの

- 普通のメッセージへの返事
- Botを呼んだ時の返事
- おみくじの結果
- 名言

## 仕様

- 追加した内容はすぐ有効になります。
- 追加先のBotとサーバーはログイン情報と選択状態から決まり、フォームでは入力しません。
- 画像はアップロードのみ対応します。ファイルパスの直接入力はできません。
- 編集、削除、詳細設定は管理画面で行います。

## 非表示にしているもの

一般メンバー向け画面では、管理者向けの詳細設定や内部項目は表示しません。
高度な条件や効果の設定が必要な場合は、管理画面側で調整してください。
