# いちよんロボ ver2.0 サーバー別機能一覧

## 目的

`/guilds/{guild_id}` を、サーバー別の機能一覧画面として扱います。
この画面では、サーバーごとの機能ON/OFF状態を確認し、権限に応じて切り替えできる土台を用意します。

既存Bot挙動、既存JSON処理、既存管理画面のDB切り替えはまだ行いません。

## 表示する機能

最初の機能一覧には以下を表示します。

* メンション反応
* 自動反応
* NGワード
* モード
* 自動投稿
* 特殊効果タグ
* 破壊

デッキ検索は最初の機能一覧には表示しません。
デッキ検索は検索型のメンション反応として、後続Phaseで「メンション反応 > 検索 > デッキ検索」に配置します。

## ON/OFF

ON/OFF状態は `feature_flags` テーブルで管理します。
未作成の `feature_key` はデフォルトONとして表示します。
切り替え時に `feature_flags` をupsertします。

このON/OFFはまず管理画面上の状態管理です。
Bot実行時の挙動へ反映するのは後続Phaseで行います。

## 権限

* `viewer`: 閲覧のみ。ON/OFFは変更できません。
* `editor`: 通常機能のON/OFFを変更できます。
* `guild_admin`: 通常機能と破壊のON/OFFを変更できます。
* `global_admin`: 全サーバーで全操作が可能です。

破壊は危険機能のため、`guild_admin` 以上のみ切り替え可能です。

## Infoモーダル

各機能行にInfoボタンを置きます。
ホバー依存にはせず、クリックでモーダルを表示します。

Infoモーダルには以下を表示します。

* 概要
* 主な設定項目
* OFF時の挙動
* 注意点

PC/スマホ両対応を前提に、簡単なHTML/CSS/JavaScriptで実装します。

## 編集リンク

各機能行には将来の編集画面への仮リンクを置きます。
リンク先の本格実装は後続Phaseで行います。
