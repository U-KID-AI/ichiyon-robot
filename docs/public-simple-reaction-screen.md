# public おみくじ追加画面

## URL

`/public`

ログイン不要で表示します。

## 対象

- Bot: いちよんロボ
- サーバー: ランセ地方
- 追加先: 既存のランダム抽選「おみくじ」

Botやサーバーの内部IDは画面やフォームに表示しません。

## できること

- 現在有効なおみくじ候補を一覧表示する
- おみくじ候補を1件追加する
- 文字だけ、画像だけ、文字と画像の組み合わせで追加する

既存のおみくじが見つからない場合は追加できません。新しい抽選は作成しません。

## 管理

編集、無効化、削除は管理画面で行います。
public画面には特殊効果などの詳細設定は表示しません。
