# 音声機能 Phase 1: VC入退室コマンド

Botへのメンション付きメッセージで、実行者がいるDiscord VCへBotを入退室させます。

## 入室

- `@Bot 入って`
- `@Bot 来て`
- `@Bot 参加`
- `@Bot vc入って`
- `@Bot ボイス入って`

実行者がVCにいない場合は、先にVCへ入るよう案内します。Botが別VCにいる場合は、実行者のVCへ移動します。

## 退出

- `@Bot 出て`
- `@Bot 抜けて`
- `@Bot 退出`
- `@Bot vc出て`
- `@Bot ボイス出て`

BotがVC未接続の場合は、その旨を返します。

## 依存関係

Discord VC接続のため `PyNaCl` が必要です。Phase 1では音声再生をしないため、`ffmpeg` はまだ必須ではありません。URL再生や音楽キューを実装する段階で、再生方式に応じて追加します。

## 手動確認

1. BotをDiscordサーバーに参加させます。
2. 実行者がVCに入ります。
3. テキストチャンネルで `@Bot 入って` を送ります。
4. Botが同じVCへ入ることを確認します。
5. `@Bot 抜けて` を送り、Botが退出することを確認します。

ログには `guild_id`、`channel_id`、`bot_instance_id` が出ます。Tokenなどの秘密値は出しません。
