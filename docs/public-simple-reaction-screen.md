# public おみくじ追加画面

## URL

`/public`

ログイン不要で表示します。

## 対象

- Bot: いちよんロボ
- サーバー: ランセ地方
- 追加先: 既存のランダム抽選「おみくじ」

Botやサーバーの内部IDは画面やフォームに表示しません。

## できること

- 現在有効なおみくじ候補を一覧表示する
- おみくじ候補を1件追加する
- 何吉、本文、画像を分けて追加する
- 本文だけ、画像だけ、本文と画像の組み合わせで追加する

既存のおみくじが見つからない場合は追加できません。新しい抽選は作成しません。

## 何吉と本文

一覧では、何吉と本文を分けて表示します。
何吉が未設定の既存候補は「何吉なし」と表示します。

追加時も、何吉と本文を別々に入力します。
保存時は既存DBの `result_label` と `body` を別々に使い、本文へ何吉を混ぜません。
Bot返信時だけ、既存ランタイムの処理で何吉と本文を組み合わせます。

## 管理

編集、無効化、削除は管理画面で行います。
public画面には特殊効果などの詳細設定は表示しません。
