# いちよんロボ ver2.0 モード設計

## 方針

はゆすモード、成田モード、しこっちモードなどを個別機能にせず、「モード」という一つの機能にまとめます。

機能一覧では「モード」をON/OFFできます。
モード機能を押すとモード一覧を表示します。
モード一覧で各モードを有効/無効にできます。
さらに各モードを押すと、そのモードの設定画面へ遷移します。

## 共通設定

各モードは以下の共通設定を持ちます。

* 継続時間
* 突入/終了通知チャンネル
* 反応チャンネル
* 非反応チャンネル

反応チャンネルが未設定なら全チャンネルで反応します。
非反応チャンネルが設定されている場合、そのチャンネルでは反応しません。

優先順位は以下です。

```txt
非反応チャンネル > 反応チャンネル > 全チャンネル
```

モード中は他のモードに遷移しない仕様にします。

## 発動方式

モードには発動方式を持たせます。

* 確率抽選型
* カウント到達型
* 手動/テスト型

## はゆすモード

確率抽選型です。

設定項目:

* 発生確率
* 再発動クールタイム
* 継続時間
* 突入/終了通知チャンネル
* 反応チャンネル
* 非反応チャンネル
* 有効/無効

現在の基本値:

* 発生確率: 1/112
* 継続時間: 180秒

## 成田モード

カウント到達型です。

対象ワード:

* お金
* データ

対象ワードを検知するたびに `:narita:` リアクションを付け、成田カウントを増やします。
22回貯まると成田モードに突入します。

毎月22日に集計期間を更新し、以下をリセットします。

* 成田カウント
* 期間内発動済み状態

1期間に1回だけ成田モードになれる想定です。

成田モード中の返答候補:

* お金の代わりにデータを持つ時代が到来する
* 稼ぐより踊れ
* まねきねこアルゴリズム
* 泥だんご
* アートークン

終了時は「寿命と預金を減らしながら元のモードに戻る」。

## しこっちモード

通常反応として「しこっち」に「しこっちきたぁぁぁ」と返します。

1/444で「しこっち、きた。」と発言し、14分疑似オフライン化します。
本当にプロセスを落とすのではなく、Botが反応しない疑似オフライン状態にします。

復帰時に「まずは女子供から殺す」と発言します。

設定項目:

* 通常反応トリガー
* 通常反応本文
* 疑似オフライン発生確率
* 疑似オフライン時間
* 疑似オフライン突入メッセージ
* 復帰メッセージ
* 有効/無効

## 疑似オフライン

疑似オフラインは、Botプロセスを停止せず、一定時間反応しない状態として扱います。

疑似オフライン中の扱い:

* 通常反応しない
* モード抽選しない
* 自動投稿など時間駆動の処理は別途安全に扱う

復帰時には復帰メッセージを投稿します。
